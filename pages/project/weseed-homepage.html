메인 화면

위시드 홈페이지 메인 페이지 입니다.

대분류로 각 서비스(ssCRM, Invest, Care, Mind)의 상세페이지, 빅마케팅, 데모/문의, 회사이야기, 새소식 등이 있습니다.

Angular 스와이퍼 라이브러리

Angular 12버전과 swiper 라이브러리가 버전이 맞지 않아 버튼이 작동하지 않았는데,

Angular 생명주기와 DOM 조작을 이용하여 정상적으로 작동하게 만들었습니다.

새소식(공지사항)

관리자 페이지에서 등록한 소식 목록을 불러와 분류 별 조회수가 가장 많은 항목만 표시합니다.

휴대폰 인증

무분별한 문의나 광고성 문의를 방지하기 위해 인증 페이지를 구성했습니다.

번호 입력 후 인증 번호 전송 시 AWS SMS를 이용하여 인증번호가 즉시 발송되며,

인증 번호 확인 시 다음 섹션으로 이동합니다.

이 때, 유효성 검사를 통하여 휴대폰 번호가 번호 형태가 아니면 인증번호를 받을 수 없고

숫자만 입력할 수 있습니다.

데모/문의 휴대폰 인증 후 정보 입력 단계

인증이 성공하면 정보 입력 페이지로 이동합니다.

장난스러운 문의를 조금이나마 필터링 하고자 reCapcha를 사용했습니다.

URL을 통한 접근 방지

인증번호를 통해 페이지 이동이 아닌 악의적인 접근(URL을 통한 접근)을 시도 할 시,

잘못된 접근이라는 팝업을 띄우며 메인 페이지로 이동합니다.

데모 체험 신청 후 계정 생성/h4>

신청 시 각 서비스(ssCRM, Invest, Mind, Care)에 데모 계정 생성을 요청하여

계정 생성 후 신청자에게 사진과 같은 내용의 메일 발송과 관리자에게는 신청이 왔다는 내용의 메일이 발송됩니다.

메일 내용의 URL에 접근하여 ID와 PWD로 데모 체험을 해볼 수 있습니다.

자세한 내용은 노션 링크에 첨부해두겠습니다 !
