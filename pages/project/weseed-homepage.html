메인 화면

위시드 홈페이지 메인 페이지 입니다.

대분류로 각 서비스(ssCRM, Invest, Care, Mind)의 상세페이지, 빅마케팅, 데모/문의, 회사이야기, 새소식 등이 있습니다.

회원가입

SNS 서비스를 이용한 회원가입 또는 이메일로 회원가입 할 수 있게끔 구성하였습니다.

로그인

회원가입 후 로그인 하여 나의 정보를 확인할 수 있습니다.

문의센터

회원 타입 별 문의 사항 페이지를 만들었습니다. 게시글 클릭 시 노션으로 이동합니다.

프로젝트 만들기

메인 페이지 상단에 프로젝트 만들기 버튼 클릭 시 프로젝트를 만들 수 있는 페이지입니다.
